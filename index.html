Open-source software (OSS) is a type of computer software in which source code is released under a license in which the copyright holder grants users the rights to study, change, and distribute the software to anyone and for any purpose.

History

In 1997, Eric Raymond published The Cathedral and the Bazaar.
The Open Source Initiative was founded in February 1998.

Movement / Ideology

The open-source-software movement is a movement that supports the use of open-source licenses for some or all software, a part of the broader notion of open collaboration.

Free Software Foundation logoRichard Stallman
The Open Source Initiative (OSI) is a California public benefit corporation, founded in 1998.

The Open Source Definition is a document published by the Open Source Initiative, to determine whether a software license can be labeled with the open-source certification mark.
The distribution terms of open-source software must comply with the following criteria:
  1. Free Redistribution
  2. Source Code
  3. Derived Works
  4. Integrity of The Author's Source Code
  5. No Discrimination Against Persons or Groups
  6. No Discrimination Against Fields of Endeavor
  7. Distribution of License
  8. License Must Not Be Specific to a Product
  9. License Must Not Restrict Other Software
  10. License Must Be Technology-Neutral
